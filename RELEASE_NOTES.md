# SliznyakVisual v2.0.9

## New Feature: Animations Module

Added a powerful new module **Animations** in the Visuals category for flexible customization of chat and tab.

### Features:
- **Chat Animations**
  - Toggle chat message animations
  - Animation styles: Slide, Fade, Slide + Fade
  - Adjustable animation speed

- **Tab (Player List) Animations**
  - Smooth pop-in animation when opening tab
  - Adjustable speed

- **Custom Chat**
  - Custom chat width
  - Chat height / length (number of visible lines)
  - Chat scale
  - Chat background opacity

### Changes:
- New module: Animations
- New mixins: ChatHudMixin, PlayerListHudMixin
- Updated Russian and English localization
- Version bumped to 2.0.9
